音视频之基础知识简单整理

发表于 2018-02-01

摘要

声音是波。

Part 8

1、声波的三要素

  • 频率、振幅和波形。频率代表音阶的高低，振幅代表着响度，波形代表着音色。

2、数字音频

  • 采样：在时间轴上对信号进行数字化【采样定理】；
  • 量化：在幅度轴上对信号进行数字化；
  • 编码：按照一定的格式记录采样和量化后的数字音频。

3、PCM

  • 音频的裸数据格式: 脉冲编码调制（Pulse Code Modulation）数据；
  • PCM描述: 量化格式，采样率，声道数;
  • 分贝是描述声音强度的单位（dB）。

4、音频编码

  • 压缩编码实际上是压缩掉冗余信号；
  • WAV编码、MP3编码、AAC编码和Ogg编码。
  • ISO制定的JPEG压缩算法

5、图像

  • 任何一个图像都可以有RGB表示，

6、视频

  • 视频帧的裸数据更多用YUV表示。

7、视频编码

  • 视频压缩是压缩冗余信息。
  • 使用帧间编码技术去除时间上的冗余信息：运动补偿、运动表示和运动估计；
  • 通过帧内编码技术去除空间上的冗余信息。
  • ISO制定的MPEG系列算法和ITU-T制定的H.26x系列算法。
  • DTS：视频解码。

8、IPB帧

  • I帧：帧内编码帧 GOP的第一帧；
  • P帧：前向预测编码帧，预测帧；
  • B帧：双向预测内插编码帧，双向预测帧。
